# Создаем базу данных на Cpp

## Введение и настройка REPL

### Функции

* ```void * malloc( size_t sizemem );``` - Функция malloc выделяет блок памяти, размером sizemem байт, и возвращает указатель на начало блока.

Содержание выделенного блока памяти не инициализируется, оно остается с неопределенными значениями.
